1. github新建一个仓库
2. 本地新建一个文件夹
3. git init 初始化下
4. git remote add origin  复制需上传的SSH地址 如：git@github.com:WZF-Anne/BlockChain-project.git     建立git连接（可放到后面进行连接）
5. 提交项目文件：
		git add .     当前目录下
		git commit -m ""     
		git push origin master   
提交到master分支下 ，也可直接到main主目录下
6. 拉取项目文件：
		git clone  直接进行git连接
	拉取代码：
		git pull origin master/main

重新提交报错，解决办法：
1、git pull origin master --allow-unrelated-histories //把远程仓库和本地同步，消除差异
2、重新add和commit相应文件
3、git push origin master
